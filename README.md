# Sanium.Web
Fragment części wizualnej aplikacji Sanium - aplikacji umożliwiającej dodawanie oraz przeglądanie ogłoszeń o pracę w dziale IT.

## Instalacja
    npm install
## Uruchomienie
    npm start
